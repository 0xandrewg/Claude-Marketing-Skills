# Content Calendar

Builds weekly content plans organized around 4 content pillars with specific formats, topics, and hook ideas for each day. Removes the "what should I post today?" decision fatigue.

## What It Does

- Creates full weekly calendars with pillar assignments, formats, and topic suggestions
- Balances the weekly mix across all 4 pillars
- Suggests topics based on current events or your input
- Fills gaps if you've already posted some days and need to round out the week

## How to Load It

1. Create a new Claude Project (or add to an existing one)
2. Go to Project Knowledge > Add content
3. Copy the entire contents of `SKILL.md` and paste it in

## How to Customize

### Replace the pillars
The defaults are Build Log, Timeline Roast, Spicy Takes, and Builder's Journal. If your content falls into different categories (e.g., "Case Studies," "Industry Analysis," "Behind the Scenes," "Hot Takes"), swap them in and update the formats and examples for each.

### Change the posting frequency
The default is 6-7 posts per week. If you post 3x/week, adjust the weekly mix rules to ensure each calendar covers at least 3 of the 4 pillars.

### Add your recurring topics
If you always post about specific products, industries, or themes, add them to the skill so the calendar draws from them.

## Example Usage

**You:** Build me a content calendar for this week. I'm building an automation for lead scoring and GTC just happened.

**Claude:** [Returns a Monday-Friday calendar with pillar assignments, format suggestions, specific angles, and hook ideas incorporating both topics]

## Works Well With

- **Post Writer**: Once the calendar tells you what to post, use the Post Writer to actually write it
- **Spicy Take Generator**: Feed Timeline Roast and Spicy Take days into this skill for the actual post
